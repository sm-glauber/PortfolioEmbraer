MUITO PRAZER, SOU GLAUBER MOURA.

Meu nome é Glauber, estou cursando Engenharia da Computação na Universidade Federal de Itajubá - UNIFEI e estou no 6° semestre. Nasci na cidade de Caçapava-SP mas atualmente moro em Itajubá-MG.

MINHAS COMPETÊNCIAS.

C

Meu primeiro contato com programação foi com a linguagem C, quando ganhei um kit iniciante de arduino, antes mesmo de iniciar na faculdade. Desde então, minha paixão e vontade por aprender mais sobre a área da tecnologia começou a crescer.

C++

Ao entrar na faculdade, deparei-me com o C++ trazendo novos desafios como a implementação de classes e a programação orientada a objeto. Com o tempo, consegui dominar a linguagem e desenvolver diversos projetos durante o curso.

Python

Logo após conhecer o C++, fui introduzido ao Python com a missão de plotar gráficos em 3D. Uma linguagem fácil de entender mas que me permitiu desenvolver projetos mais complexos e aprender sobre a área de inteligência artificial desenvolvendo alguns projetos de machine learning.

JAVA

Durante o curso, também tive a oportunidade de aprender a linguagem Java, que abriu portas para o conhecer novas linguagens. Com ela, desenvolvi projetos de banco de dados e aprendi a trabalhar com persistência de classes.

SQL

Com o Java, também aprendi a trabalhar com banco de dados, utilizando a linguagem SQL para criar e manipular tabelas. Com o SQL, fui capaz de desenvolver projetos utilizando diversos tipo de bancos de dados: relacionais, orientado a objetos, orientado à grafo e orientado a documentos (MySQL, MariaDB, Neo4j, MongoDB).

Assembly

Recentemente, conheci o Assembly, uma linguagem de baixo nível que me permitiu entender melhor o funcionamento de processadores e manipulação de registradores. Tive a oportunidade de utilizar a linguagem em processadores ARM e RiscV.
